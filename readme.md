

## Project Description
This project detects whether an email is Spam or Not Spam using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes

## How to Run
1. Install the required libraries:
   ```bash
   pip install pandas scikit-learn
   ```
2. Run the program:
   ```bash
   python spam_classifier.py
   ```
